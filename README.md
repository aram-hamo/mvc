## Description
Minimal MVC Framework written in PHP.

## Getting Started
### Installation
```
composer create-project aram-hamo/mvc
```
### Starting the local dev server
The default port is ```8080```.
```
php mvc serve
```
### DB Migration
This will migrate all the migration files found in ```migration/``` directory.
```
php mvc migrate
```
